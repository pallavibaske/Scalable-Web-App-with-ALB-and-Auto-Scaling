# 🚀 Scalable Web App with ALB and Auto Scaling

## 🎯 Project Overview

This project demonstrates how to deploy a highly available and scalable web application on AWS using Amazon EC2, Application Load Balancer (ALB), and Auto Scaling Group.

The architecture automatically distributes incoming traffic across multiple EC2 instances and launches or terminates instances based on traffic demand.

---

## 🛠 AWS Services Used

* Amazon EC2
* Application Load Balancer (ALB)
* Auto Scaling Group (ASG)
* Security Groups
* Target Group
* Amazon VPC
* Internet Gateway

---

## 🏗️ Architecture

The application is deployed on multiple EC2 instances behind an Application Load Balancer. Auto Scaling automatically adjusts the number of EC2 instances according to incoming traffic.

---

## 🔄 Workflow

1. User accesses the Application Load Balancer.
2. ALB distributes traffic to healthy EC2 instances.
3. Auto Scaling monitors the workload.
4. New EC2 instances are launched automatically when traffic increases.
5. Instances are terminated automatically when traffic decreases.
